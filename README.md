SushiTerm
=========

A tool that allows you to practice typing on the terminal like [寿司打](http://typingx0.net/sushida/).

![実行画面](./image/exec.png)

## Requirement
- Ruby


## Usage
### Play Game
```sh
$ ruby sushiterm.rb
```

## Installation
```sh
$ git clone https://github.com/AtLab-jp/SushiTerm
$ cd SushiTerm
```
